Glance Vessels
==============

This is built off of ParaView Glance.

Building
========

```
$ git clone git@github.com:KitwareMedical/glance-vessels.git
$ cd glance-vessels/
$ npm install
$ cd server/
$ pip install -r requirements.txt
```

Running
=======

One terminal:
```
$ npm run dev
```

Other terminal:
```
$ python server.py
```
